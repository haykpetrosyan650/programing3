# Գույն Նկարագրություն
1 Կանաչը՝ հայաբնակ (ոչ՜ Հայաստանը) տարածքներն են,որոնք բազմանում են ազատ տարածքներում, և մահանում է երբ նրան ուտում (գրավում) է Ադրբեջանը, կամ Ռուսաստանը:
2 Դեղինը՝ Ադրբեջանն է, որը բազմանում է N հատ հայաբնակ տարածք ուտելիս (գրավելիս), կամ Ռուսաստանին ուտելիս(գրավելիս), և մահանում է N հատ չուտելուց հետո, կամ երբ նրան ուտում է Ռուսաստանը:
3 Սպիտակը՝ Ռուսաստանն է, որը բազմանում է N հատ հայաբնակ տարածք ուտելիս (գրավելիս), կամ Ադրբեջանին ուտելիս(գրավելիս), և մահանում է N հատ չուտելուց հետո, կամ երբ նրան ուտում է Ադրբեջանը :
4 Կարմիրը՝ Թուրքիան է, որը բազմանում է N հատ Ռուսաստանի տարածքը ուտելիս (գրավելիս), կամ Ադրբեջանն միավորվելիս:
5 Կապույտը՝ Հայաստանն է, որը բազմանում է N հատ Ադրբեջանի տարածքը ուտելիս (գրավելիս), կամ հայաբնակ տարածքներն միավորվելիս:
